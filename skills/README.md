# Lauren MCP Skills

Skills are self-contained, copy-paste-ready guides for the most common `lauren-mcp`
tasks. Each skill lives in its own directory and contains a `SKILL.md` file.

| Skill | Directory | Description |
|---|---|---|
| Using MCP Server | `using-mcp-server/` | Expose a Lauren service with `@mcp_server` and `@mcp_tool` |
| Using MCP Client | `using-mcp-client/` | Connect to a remote MCP server over stdio, WebSocket, or HTTP+SSE |
| MCP Agent Tools | `mcp-agent-tools/` | Wire MCP server tools into a Lauren `AgentModule` |
| MCP Testing | `mcp-testing/` | Test MCP servers with the echo server pattern and protocol mocks |
| MCP Transport Internals | `mcp-transport-internals/` | Understand `McpDispatcher`, `SseSessionStore`, and base client internals |
